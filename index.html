Creating My Heading with Color

When you create your heading, you need to use the heading elements such as h1, h2, h3, up to h6, and make sure you have both the opening and closing tags. You can make your heading have some color that you like using the css style.

CSS Style

  • Pick a selector (h1)
  • Create a declaration block
  • Pick a property (color)
  • Select a value (blue)
